.. function:: getEleLoadData <patternTag?>

   This function returns the load data for a given pattern tag. The load data is
   returned as a list of floats.

   :param patternTag: The pattern tag for the load data (optional).
   :type patternTag: int
   :return: A list of floats containing the load data.
   :rtype: list
